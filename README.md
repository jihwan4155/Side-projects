### 📝 학습 로그 (Daily Logs)

매일 작업한 내용과 고민한 흔적은 각 프로젝트 폴더 안의 `log.md` 파일에 기록합니다.

1. **오늘의 목표**: 작업 시작 전, 구체적으로 완료할 기능을 정의합니다.
2. **진행 상황**: 체크박스(`- [x]`)를 활용해 구현된 기능을 시각화합니다.
3. **오늘의 도전 & 해결 (TIL)**: 
   - 발생한 에러 메시지와 해결 과정을 기록합니다.
   - 새롭게 배운 라이브러리나 파이썬 문법을 정리합니다.
4. **내일의 할 일**: 다음 단계의 작업 흐름을 미리 구상하여 기록합니다.
